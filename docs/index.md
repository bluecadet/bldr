# :muscle: Bldr

Bldr is a (very opinionated) configurable task runner for css, js, and sass with a local development via Browsersync.

## One Config to Rule Them All

Bldr runs one configuration to handle multiple asset sources and destinations. Multiple sources, multiple destinations.

## What's it for?

Bldr is not for every project. Bldr is at home in:

- :thumbsup: php applications like Drupal, WordPress, or Craft
- :thumbsup: basic html sites
- :thumbsup: places where you just wanna write modern css/js and have it transformed

## What's it not for?

- :thumbsdown: React
- :thumbsdown: Vue
- :thumbsdown: Other similar js framework kin


## Processing Providers
- PostCSS
- Dart Sass
- EsBuild (dev only)
- Rollup (build only)
- EsLint
- Stylelint
- Biome
